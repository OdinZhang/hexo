要想使用sympy包，肯定是要先导入，下面是导入的两种方法

import sympy

import sympy as *

这两者之间的差距肯定是有的，第一种的话就是在使用sympy函数的时候需要在其前面加一个sympy.以示其位于sympy包内，第二种方法则不需要。不过如果导入多个包的话建议用第一种方法以示区分，否则可能会有一些函数冲突

在sympy中，要想使用一个字母（例如x作为变量），需要预先声明

x = symbols('x')

多个变量的话则可以用

x,y = symbols('x y')

作为简写以提升输入速度

在notebook中输入一段简单的代码如下

from sympy import *
x=symbols('x')
expand(x*(x+1))

这样的话输出的结果是

x**2 + x

这个样子，但是这并不符合我们的日常使用规范，因此，只需要在第二行添加

init_printing(use_unicode=True)

或者

init_printing()

就能输出我们日常所习惯的样式，如下

$x^{2}+x$

当然，有些时候我们并不想输出展开的形式，那么只需要把第三行改为

x*(x+1)

就可以了，当然，使用函数factor()也是可以强制转换为$x(x+1)$的形式

接下来将介绍sympy里面几个比较常用的函数

  1. 求导（以cos(x)）为例

diff(cos(x),x)

  1. 不定积分（以x+1为例）

integrate(x+1,x)

  1. 定积分（以x+1在0到1上的积分为例）

integrate(x+1,(x,0,1))

  1. 极限（以1/x其中x趋于正无穷为例）

limit(1/x,x,+oo)

  1. 解方程（以x^2-2=0为例）

solve(x**2 - 2, x)

  1. 解微分方程（以y’’-y=e^t为例）

    y = Function(‘y’)
    dsolve(Eq(y(t).diff(t, t) - y(t), exp(t)), y(t))
